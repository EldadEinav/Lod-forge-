# LOD Forge

Real-time browser-based mesh preview starter for OBJ / GLTF / GLB.

## Quick Start

1. Upload `index.html` to your repository root
2. Enable GitHub Pages from `main` branch
3. Open the generated site URL
4. Drop a mesh file into the tool
